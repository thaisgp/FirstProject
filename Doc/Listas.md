Utilize (*) para inserir itens em lista
` * item 1`
  * item 1

Um subitem é feito de maneira semelhante
```
* item 1 
    * subitem
```
Obtendo: 

* item 1 
    * subitem

É possível fazer muitos subitens seguindo o mesmo exemplo .

* item 1 
    * subitem
        * subitem
            * subitem
                * subitem    
                    * subitem
                         * subitem
                             * subitem   
                                 * subitem
